# Sentinel Watch V14 Web App Package

This package is a host-ready installable web app (PWA).

Files:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- apple-touch-icon.png

How to use:
1. Upload all files to a static web host such as GitHub Pages, Netlify, or Vercel.
2. Open the hosted URL in Safari or Chrome.
3. On iPhone, use Share → Add to Home Screen.

Notes:
- This is an installable command-center shell and mobile web app.
- It is not a true live API ingestion platform.
- Source buttons open external live sites and support copy-url fallback.
